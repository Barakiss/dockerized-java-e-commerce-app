Τα Λαχανικά και τα Φρούτα στη καθημερινοτητα

Τα φρούτα και τα λαχανικά πρέπει να αποτελούν σημαντικό μέρος της καθημερινής διατροφής σας. Είναι φυσικά και περιέχουν βιταμίνες και μέταλλα που μπορούν να σας βοηθήσουν και να σας κρατήσουμε υγιείς. Μπορούν επίσης να βοηθήσουν στην προστασία από πολλές ασθένειες.

Οι περισσότεροι Έλληνες θα επωφεληθούν από την κατανάλωση περισσότερων φρούτων και λαχανικών ως μέρος μιας ισορροπημένης, τακτικής διατροφής και ενός υγιούς, ενεργού τρόπου ζωής. Υπάρχουν πολλές ποικιλίες φρούτων και λαχανικών διαθέσιμες και πολλοί τρόποι για να προετοιμάσετε, να τα μαγειρέψετε και να τα σερβίρετε.

Θα πρέπει να τρώτε τουλάχιστον πέντε διαφορετικά λαχανικά και δύο φρούτα κάθε μέρα. Επιλέξτε διαφορετικά χρώματα και ποικιλίες.

Μια μερίδα λαχανικών είναι περίπου ένα πιάτο ωμών ή μισό πιάτο μαγειρεμένων.

Μια μερίδα φρούτων είναι περίπου ένα μεσαίου μεγέθους ή δύο μικρά κομμάτια από φρούτο κομπόστα (χωρίς πρόσθετα σάκχαρα).
